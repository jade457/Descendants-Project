User Stories

Need-to-have features

As a student enrolled in ANTH370, in order to better make use of the disparate documents, I can use this tool to visually build a family tree and organize documents by associating them with nodes in the tree

As a student enrolled in ANTH370, in order to store information for a particular family, I can create a "profile" for that family and work on constructing the family tree in that profile.

As a student enrolled in ANTH370, in order to access the product, I can enter a class key that will give me access to the shared materials for the classroom.

As a family member of the victim, in order to see my family tree, I can access existing family trees, zoom in and out, and access the documents without needing controls for edits.

Nice-to-have features

As a student enrolled in ANTH370, in order to work on my family tree, I can access it via a login interface.

As a student enrolled in ANTH370, to share my family tree, I can choose to make it public so that anyone can access the tree or to make it private, requiring credentials to view the tree.

As a family member of the victim, in order to share my family tree, I can export it as a pdf/png/jpg file to print out.

As a student enrolled in ANTH370, in order to view the family tree in an alternative way, I can change the view from a visual tree to a list format, where indenting may indicate a descendant relationship.
